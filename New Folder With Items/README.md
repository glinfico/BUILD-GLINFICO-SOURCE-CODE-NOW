# GLINFICO V2

Vercel-ready Next.js starter for GLINFICO with:
- marketing pages
- auth shells
- MCA / REI / M&A application flows
- client / broker / admin dashboards
- mock lender matching
- Twin AI webhook prototype
- Stripe webhook placeholder

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Upload this folder to GitHub.
2. Create a new Vercel project from the repository.
3. Add environment variables from `env.example`.
4. Deploy.

## Notes

This is a production-style starter, but still uses mock data and placeholder API logic. Next steps are Supabase auth/database, Stripe checkout, document uploads, and real Twin AI integration.
